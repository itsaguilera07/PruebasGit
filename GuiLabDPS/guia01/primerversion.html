Parrafo 1

Git se diseñó para gestionar de manera eficiente y confiable el control de versiones en proyectos con una gran cantidad de archivos de código fuente. En esencia, Git proporciona las herramientas necesarias para que los equipos desarrollen software o páginas web de forma colaborativa, inteligente y rápida.


"Los Sistemas de Control de Versiones Distribuidos (DVCS) resuelven eficazmente esta problemática. Además de Git, existen otras herramientas distribuidas como Mercurial, Bazaar o Darcs. Una característica clave de estos sistemas es que cada cliente obtiene una copia completa del repositorio."
